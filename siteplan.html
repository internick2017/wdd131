Website Planning Document

Site Name

DevPortfolio - A professional web development portfolio and resume site.

I chose this name because it clearly communicates the site's purpose to visitors. It combines "Dev" (short for developer) with "Portfolio" to create a memorable and descriptive name that highlights my focus on web development. The name is concise, professional, and directly relates to the content visitors will find on the site.

Optional domain availability: devportfolio.io or mydevportfolio.com

Site Purpose

The purpose of this site is to showcase my web development skills, projects, and professional background to potential employers, clients, and fellow developers. The site will serve as a:

  • Professional portfolio featuring completed web development projects with descriptions, screenshots, and links
  • Digital resume highlighting my technical skills, education, and work experience
  • Contact hub where interested parties can reach me for employment or project opportunities
  • Platform to demonstrate my coding abilities through the site's own implementation and design

Scenarios

Target Audience: Potential Employers, Clients, and Fellow Developers

  1. Scenario 1: "As a hiring manager at a web development agency, what kinds of projects has this developer completed, and what specific technologies do they have experience with?"

  2. Scenario 2: "As a potential client looking for a freelance web developer, how can I contact this person to discuss my project needs and get a quote?"

  3. Scenario 3: "As a fellow developer, what is this person's coding style and approach to problem-solving in web development projects?"

Color Application

  • Primary Color (Blue #3498db): Navigation links, buttons, section headings, and other interactive elements
  • Secondary Color (Dark Blue/Gray #2c3e50): Header background, footer background, secondary headings
  • Accent Color (Red #e74c3c): Call-to-action buttons, important text highlights, and key information emphasis
  • Background Color (Light Gray #f8f9fa): Page background and content areas
  • Text Color (Dark Gray #333333): Primary body text
